
#  Rose Day 

A handcrafted, cinematic Rose Day website designed to feel **soft, warm, and emotionally intimate**.

This project is not a template or a demo it is a **story told through light, motion, and silence**. Every scroll reveals a new emotional beat, guided by a living 3D rose that responds gently to the viewer.

---

##  Concept

The idea behind this project was simple:

> A rose should not just be seen it should be *felt*.

Instead of loud animations or flashy UI, this experience focuses on:

* Calm pacing
* Candlelight warmth
* Subtle motion
* Emotional presence

---

##  Key Features

### Cinematic Scroll Direction

* Scroll driven choreography (no random motion)
* Rose transitions through emotional "poses"
* Smooth inertia-based movement for a heavy, premium feel

### 3D Rose (Three.js)

* Custom `rose.glb` model
* Velvet / porcelain-inspired material
* Soft sheen and warm edge highlights
* Responsive scaling for mobile, tablet, and desktop

### Romantic Atmosphere

* Warm candlelight color grading
* Soft fog and vignette (never pure black)
* Emotion-first lighting (ambient, key, rim, fill)

### Floating Petals & Hearts

* Canvas-based particle system
* Depth-aware motion (foreground / background)
* Petals and hearts behave like drifting thoughts
* Optimized count for mobile and desktop

### Elegant Typography

* Serif-led romantic typography
* Whisper-style text reveal animation
* Spacious, breathable layout

### Sound (Optional)

* Soft ambient background music
* User-controlled (mobile-safe)
* Gentle fade-in / fade-out

---

##  Fully Responsive

The experience is carefully tuned for:

* iPhone (portrait & landscape)
* iPad / tablets
* MacBook & desktop screens

Responsive adjustments include:

* Camera distance & framing
* Rose position per section
* Particle count optimization
* Touch-based parallax on mobile
