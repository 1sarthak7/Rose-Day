##  Key Features

### Cinematic Scroll Direction

* Scroll-driven choreography (no random motion)
* Rose transitions through emotional "poses"
* Smooth inertia-based movement for a heavy, premium feel

### 3D Rose (Three.js)

* Custom `rose.glb` model
* Velvet / porcelain-inspired material
* Soft sheen and warm edge highlights
* Responsive scaling for mobile, tablet, and desktop

### Romantic Atmosphere

* Warm candlelight color grading
* Soft fog and vignette (never pure black)
* Emotion-first lighting (ambient, key, rim, fill)

### Floating Petals & Hearts

* Canvas-based particle system
* Depth-aware motion (foreground / background)
* Petals and hearts behave like drifting thoughts
* Optimized count for mobile and desktop

### Elegant Typography

* Serif-led romantic typography
* Whisper-style text reveal animation
* Spacious, breathable layout

### Sound (Optional)

* Soft ambient background music
* User-controlled (mobile-safe)
* Gentle fade-in / fade-out
